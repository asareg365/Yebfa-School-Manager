# **App Name**: Yebfa School Manager

## Core Features:

- Multi-Tenant Partitioning: Secure Firestore-backed architecture providing complete data isolation and tenantId-based filtering for distinct school instances.
- Onboarding Wizard: A streamlined school registration flow capturing motto, logo, and academic metadata with automatic unique Tenant ID generation.
- Identity & Access Hub: Integrated role-based permission system supporting eight distinct school personas for precise administrative and portal access control.
- Integrated Academic Ledger: Centralized record system for student enrollment, faculty subject assignment, and multi-stream class management.
- Smart Attendance Insights: Bulk-update attendance trackers that calculate term-based percentages and trigger automated absence alerts via Firebase Cloud Messaging.
- AI Academic Narrative Tool: A generative tool that analyzes student exam scores and attendance data to reason and suggest personalized performance report comments.
- Fintech Billing Engine: Automated invoice generation for fees, payroll processing for staff, and real-time expense reconciliation.
- AI Financial Forecasting Tool: A tool that uses reasoning to analyze revenue trends and expense patterns to provide predictive insights for school budgeting.

## Style Guidelines:

- Primary: Midnight Blue (#1E293B) for a deep, trustworthy institutional foundation.
- Background: Ice White (#F8FAFC) to provide a clean, high-contrast workspace.
- Accent: Sunrise Orange (#F59E0B) for high-visibility call-to-action buttons and notifications.
- Headlines use 'Space Grotesk' for a modern geometric feel, while 'Inter' is used for body text to ensure maximum readability in data-heavy views.
- System-wide use of Lucide icons with a 1.5px stroke for a clean, professional aesthetic.
- Enterprise-grade dashboard layout with a collapsible sidebar and card-based modular grids inspired by Notion.
- Subtle slide-in transitions for side panels and soft fade-ins for dashboard widget loading states.
